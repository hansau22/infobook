Class DatabaseHandler

source code

Infolib.DatabaseHandler ist eine Klasse, die den Datenbankzugriff auf eine sqlite3-Datenbank fuer den Infobook-Server regelt.

Eine Verwendung ausserhalb des Infobook-Projektes wird nicht empfohlen.
