Class ConnectionHandler

source code

Infolib.ConnectionHandler ist eine Klasse, die die Grundfunktionalitaet des Infoservers bereitstellt. Sie bindet einen Port, an dem alle Verbindungen eigehen und verteilt die Pakete nach den entsprechenden Kopfinformationen. Sie baut auf allen anderen Klassen in der Infolib auf.

Eine Verwendung ausserhalb des Infobook-Projektes wird nicht empfohlen.

Eine Dokumentation des Ablaufes finden Sie im GitHub-Wiki unter https://github.org/hansau22/infobook/
