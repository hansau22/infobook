Class EncryptionHandler

source code

Infolib.EncryptionHandler ist eine Klasse, die Funktionen zur Ver- und Entschluesselung von Strings bereithaelt.

Eine Verwendung ausserhalb des Infobook-Projektes wird nicht empfohlen.

Es werden AES mit einem 265-Bit key im CTR-Modus als symmetrische Verschluesselung und SHA256 als Hashaglgorithmus verwendet.
